# BEnito

Proyecto que une la interfaz de CraftHub con el backend separado de bakendcraft para trabajar de forma conjunta y publicarlo en GitHub.

## Estructura

- CraftHub/: aplicación principal de la interfaz.
- bakendcraft/: módulos backend organizados por pantalla.

## Requisitos

- Python 3.10+
- Dependencias de la app según el entorno de desarrollo

## Uso

1. Clona este repositorio.
2. Instala las dependencias necesarias.
3. Ejecuta la aplicación principal desde CraftHub.

## Notas

El backend se mantiene separado por módulos, pero ahora forma parte del mismo repositorio para facilitar su gestión y despliegue.
